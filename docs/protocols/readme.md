# Протоколы

Этот каталог содержит актуальные спецификации протоколов и вспомогательные материалы.

## Актуальные документы
- `PROTOCOL_TK_ETHERCAT.md` — основной протокол обмена с ТК (EtherCAT PDO).
- `PCCOM4.02.md` — протокол PCcom4 (USB-UART, плата ↔ ПК).
- `PCCOM4.02_PROJECT.md` — проектные расширения/профиль PCcom4.
- `obsolete/CAN_LOGGING_HOWTO_RU.md` — как снимать логи CAN/CAN FD (исторические случаи, устар.).

## Устаревшие документы
Папка `obsolete/` содержит устаревшие документы по legacy CAN *(устар.)*. Использовать только как историческую справку.
