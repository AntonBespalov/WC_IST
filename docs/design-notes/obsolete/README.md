# Obsolete Design Notes

В этой папке хранятся **устаревшие** Design Notes (DN).

Критерии для переноса сюда:
- документ заменён новым DN / ADR,
- предпосылки изменились и описанное решение больше не применяется,
- документ признан незначимым для текущей системы (оставлен только для истории/аудита).

Требование: в устаревшем документе указывать причину устаревания и ссылку на актуальный документ (если есть).
