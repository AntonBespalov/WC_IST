# Obsolete ADR

В этой папке хранятся **устаревшие** ADR (Architecture Decision Records).

Критерии для переноса сюда:
- решение заменено новым ADR / Design Note,
- предпосылки изменились и решение больше не применяется,
- документ признан незначимым для текущей архитектуры (оставлен только для истории/аудита).

Требование: в устаревшем документе указывать причину устаревания и ссылку на актуальный документ (если есть).
