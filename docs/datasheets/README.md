# docs/datasheets/ — даташиты и справочные PDF

Цель: держать в репозитории **локальные копии** даташитов/errata/manual, которые нужны для разработки, ревью и верификации.

## Правила поддержки
- Структура: группировать по микросхемам/поставщикам/подсистемам (папками).
- Имена файлов: по возможности сохранять “как у производителя” (чтобы легче было сопоставлять с ревизией).
- Обновления: при замене PDF стараться сохранять историю изменения в PR-описании (что обновлено и почему).
- Ссылки из документов: если документ опирается на конкретную микросхему, допустимо ссылаться на PDF в этом каталоге вместо внешней ссылки.

## Содержимое (неполный список)
- `stm32g474/` — STM32G474: datasheet, reference manual, errata.
- `ad7380/` — AD7380.
- `ad7606/` — AD7606.
- `psram/` — APS6404L (PSRAM).
- `watchdog/` — CBM706T.
- `skyper 42 r/` — SKYPER 42 R.
- `EtherCAT/` — COMX и связанные документы.
